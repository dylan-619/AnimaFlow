# 测试短片配置适配后端规范

## 数据模型对照

### t_project 表字段映射

| 配置项 | 后端字段 | 前端表单 | 示例值 |
|--------|---------|----------|--------|
| 项目名称 | `name` | form.name | 晨光咖啡 |
| 项目类型 | `type` | form.type | 都市 |
| 美术风格 | `artStyle` | form.artStyle (多选) | 动漫,写实风格 |
| 视觉风格指引 | `styleGuide` | form.styleGuide (textarea) | 色调：暖色调... |
| 视频比例 | `videoRatio` | form.videoRatio (radio) | 16:9 |
| 项目简介 | `intro` | form.intro (textarea) | 一段发生在清晨... |

---

## 配置文件结构规范

### 1. 【项目配置】区块

必须包含的字段：

```
项目名称：对应 t_project.name
项目类型：对应 t_project.type
美术风格：对应 t_project.artStyle（逗号分隔）
视觉风格指引：对应 t_project.styleGuide（分号分隔的多行）
视频比例：对应 t_project.videoRatio
项目简介：对应 t_project.intro
```

**示例**：
```
项目名称：晨光咖啡
项目类型：都市
美术风格：动漫,写实风格
视觉风格指引：色调：暖色调...；光影：侧逆光...；构图：黄金分割...
视频比例：16:9
项目简介：一段发生在清晨咖啡店的温馨对话...
```

---

### 2. 【资产设定】区块

适配后端 `t_assets` 表字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| name | 资产名称 | 林晨 / 晨光咖啡店 / 咖啡杯 |
| type | 资产类型 | role（角色）/ scene（场景）/ props（道具） |
| intro | 详细描述 | 外貌、服装、环境描述 |
| prompt | AI 提示词 | 创作提示词库中的内容 |
| voiceType | 声音定位（仅角色） | 青年男声（温和沉稳） |
| defaultEmotion | 默认情绪（仅角色） | 平和，略带疲惫 |

**示例**：
```
角色：林晨
类型：role
外貌：男，外表28岁...
服装：灰色西装外套...
声音定位：青年男声（温和沉稳）
默认情绪：平和，略带疲惫
```

---

### 3. 【创作提示词库】区块

为 AI 生成准备的提示词，适配 `t_assets.prompt` 字段。

**格式**：
```
角色/场景/道具名称：提示词内容
```

**示例**：
```
角色-林晨：动漫风格，赛璐璃上色，大眼比例...
场景-晨光咖啡店：日系咖啡店，温馨风格...
道具-咖啡杯：白色陶瓷马克杯，几何图案...
```

---

### 4. 【创作流程建议】区块

引导用户按正确顺序操作，适配前端工作流：

1. 创建项目 → 填写配置信息
2. 导入小说 → 复制正文到编辑器
3. 生成大纲 → AI 自动生成分集
4. 提取资产 → 系统自动提取，手动补充
5. 资产生成 → 使用提示词库
6. 剧本生成 → 基于大纲生成
7. 分镜拆解 → 拆解为镜头
8. 分镜图生成 → 使用视觉风格配置
9. 配音生成 → 根据说话者和情绪
10. 视频生成 → 生成分镜视频
11. 视频合成 → 最终成片

## 前端表单适配

### 新建项目表单

```vue
<el-form-item label="项目名称">
  <el-input v-model="form.name" placeholder="项目名称" />
</el-form-item>

<el-form-item label="项目类型">
  <el-select v-model="form.type">
    <el-option label="仙侠" value="仙侠" />
    <el-option label="都市" value="都市" />
    <!-- ... -->
  </el-select>
</el-form-item>

<el-form-item label="美术风格">
  <el-select v-model="form.artStyle" multiple filterable allow-create>
    <!-- 分组选项 -->
  </el-select>
</el-form-item>

<el-form-item label="视觉风格">
  <el-input v-model="form.styleGuide" type="textarea" :rows="4"
    placeholder="详细视觉风格指引（色调、光影、镜头风格、特效标识等）" />
</el-form-item>

<el-form-item label="视频比例">
  <el-radio-group v-model="form.videoRatio">
    <el-radio-button label="16:9">16:9 横屏</el-radio-button>
    <el-radio-button label="9:16">9:16 竖屏</el-radio-button>
    <el-radio-button label="1:1">1:1 方形</el-radio-button>
    <el-radio-button label="4:3">4:3 经典</el-radio-button>
  </el-radio-group>
</el-form-item>

<el-form-item label="项目简介">
  <el-input v-model="form.intro" type="textarea" :rows="3"
    placeholder="项目简介/世界观" />
</el-form-item>
```

## 数据提交格式

### 创建项目 API 请求

```json
{
  "name": "晨光咖啡",
  "type": "都市",
  "artStyle": "动漫,写实风格",  // 多选转为逗号分隔
  "styleGuide": "色调：暖色调为主...；光影：侧逆光为主...；构图：黄金分割...",
  "videoRatio": "16:9",
  "intro": "一段发生在清晨咖啡店的温馨对话..."
}
```

## 资产创建适配

### 角色资产（role）

```json
{
  "name": "林晨",
  "type": "role",
  "intro": "男，外表28岁。动漫风格，大眼比例...",
  "prompt": "动漫风格，赛璐璃上色，大眼比例，温和表情，眼镜，黑色短发...",
  "voiceType": "青年男声（温和沉稳）",
  "defaultEmotion": "平和，略带疲惫"
}
```

### 场景资产（scene）

```json
{
  "name": "晨光咖啡店",
  "type": "scene",
  "intro": "一家温馨的小型咖啡店，日系风格...",
  "prompt": "日系咖啡店，温馨风格，原木色调，大面积落地窗..."
}
```

### 道具资产（props）

```json
{
  "name": "咖啡杯",
  "type": "props",
  "intro": "白色陶瓷马克杯，杯身有简约的几何图案...",
  "prompt": "白色陶瓷马克杯，几何图案，热气腾腾，特写镜头，4K高清"
}
```

## 验证清单

创建测试短片项目时，确保：

- [ ] 项目配置映射到 t_project 所有字段
- [ ] 美术风格使用逗号分隔的多选格式
- [ ] 视觉风格指引使用分号分隔的多行格式
- [ ] 视频比例为 16:9 / 9:16 / 1:1 / 4:3 之一
- [ ] 资产设定包含 type 字段（role/scene/props）
- [ ] 角色资产包含 voiceType 和 defaultEmotion
- [ ] 创作提示词库包含所有资产的 prompt 内容
- [ ] 正文内容可直接复制到小说编辑器
- [ ] 创作流程建议与前端操作流程一致

---

## 注意事项

1. **字段命名一致**：配置文件标签必须与后端字段名称对应
2. **数据格式规范**：
   - artStyle：多选数组 → 提交时转为逗号分隔字符串
   - styleGuide：多行文本 → 分号分隔
   - videoRatio：固定枚举值
3. **资产类型区分**：role / scene / props 必须正确标识
4. **可选字段**：旁白作为特殊角色，type 为 role 但无实体图像
5. **提示词复用**：创作提示词库中的内容直接用于资产生成
